File, Folder, and Image Properties Dialog Boxes

 

          All properties dialog boxes have been designed to look similar to the properties dialog boxes displayed by Windows Explorer.

 

1.     File properties dialog box: The file properties dialog box displays statistical data about a file. It is shown in the following screenshot:

               

The General Properties tab lists general file properties like, filename, file type, and name of the associated program, location of file, dates, and file attributes. The Advanced Properties tab lists the extended properties about the file, if any are available. Note that you cannot make any changes to the file properties.

 

2.  Folder properties dialog box: The folder properties dialog box displays statistical data about a folder:

The dialog box displays information about the selected folder such as its location, size, no. of files, no. of subfolders etc.

 

3.  The image properties dialog box: The image properties dialog box displays statistical data about an image. It also allows you to change several image properties, such as its name, category etc.

          

The General Properties tab displays general properties about the image such as the image name, image description, category, and image picture. The Advanced Properties dialog box provides detailed statistical information about the image. The image source is displayed at the top. You can change the source media of the image by clicking the Change button. The next section describes the details about the source media such as the source drive type (Fixed/Network/Removable etc.), the source file system, and the volume label of the source media. The next section displays the name of the database file of this image, total no. of files in this image, total number of directories in the image as well as the size of the database. The final section displays the creation date, modification date, and the access date of the image.
