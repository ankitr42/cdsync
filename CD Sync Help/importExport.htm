Importing and Exporting Images

 

The Export Images and Import Images commands available in the Tools menu are used to export and import images to and from a folder. The images can be exported for backup purposes, or simply to move from one PC to another.

 

1.     Exporting Images: To export images, select Tools > Export Images option. This will present you with the Export Images dialog box. To export images, follow these steps while the Export Images dialog box is displayed:

a)     Click the Browse button and select the destination folder.

b)     Once you have selected the destination folder, select the images that you would like to be exported. If you click on an image node, the details of that image will be displayed in the Image Properties section below the list box.

c)     Once you have selected the images to export, click the Export button to begin the export process.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

2.     Importing Images: To import images, select Tools > Import Images option. This will present you with the Import Images dialog box. To import previously exported images, follow these steps while the Import Images dialog box is displayed:

a)     Click the Browse button to select the folder that contains the images you want to import. Once you select a folder, it will be scanned for images, and a list of images found eligible for import will be displayed in the box below the Browse button in a hierarchical, tree like structure.

b)     The images belonging to categories that already exist will be categorized under the Existing Categories node. Similarly, images that belong to categories that do not yet exist will be categorized under the Non-Existent Categories.

c)     Like the Export Images dialog box, if you click on an image node, then details of the corresponding image will be shown in the Image Properties section below the list box. Select the images that you would like to be imported.

d)     Once you have selected the images to import, click the Import button to begin the import process.

e)     If you had selected some images belonging to the Non-Existent categories group, then you will be prompted to create those categories during the import process.

f)      When the import process completes, the image list in the main window will be updated to show the newly imported images.
