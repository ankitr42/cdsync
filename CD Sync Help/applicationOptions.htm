Configuring Application Options

 

The Tools > Options command brings up the configuration dialog box. You can also press the keyboard shortcut Ctrl+O. The Options dialog box contains three tabs, namely General, User Interface, and User Interface – Details View. These tree tabs are explained below:

 

1.     The General tab: The general tab contains options to configure the directories used by the program and the notifications that the program issues. This tab contains the following settings:

a)     Store images in: This textbox contains the path of the directory where CD Sync Portable stores its image database. This is a read only option and cannot be changed. The images that you create are stored in the Database subdirectory inside the application installation directory.

b)     Default backup path: This textbox specifies the default backup folder. This setting is used by the Export Images dialog box.

c)     Default restore path: This textbox specifies the default restore folder. This setting is used by the Import Images dialog box.

d)     Notify me if some images belong to deleted categories: This option itself is self-explanatory. If this checkbox is selected and there are images that belong to non-existent categories, then program will remind you of the same from time to time.

 

 

 

 

 

 

 

 

2.     The User Interface tab: The user-interface tab contains options to modify the appearance and behavior of the interface of the program. This tab contains the following settings:

a)     Hide files and folders with any of these attributes: This setting has six check boxes, each denoting a file attribute. Selecting an attribute here causes any files and folder with that attribute to be hidden in the File Browser. This applies to all the images. For ex., if you select have selected Hidden and Compressed, files and folders with any or both of these attributes will not be shown in the file browser.

a)     Show count of invisible objects on the bottom panel: If this option is selected, then the number of file and/or folders that are hidden from view will be displayed on the file browser information area. See User Interface to find where the file browser information area is.

b)     Show file extensions: If enabled, file extensions are also displayed by the File Browser.

c)     Show hidden and system files/folders with faded icons: If enabled, this option causes the File Browser to use “faded” icons (like the ones used by Windows Explorer) to represent files/folders that have either or both hidden and system attribute set. Otherwise, such files/folders are displayed like normal file/folders.

d)     Show details of selected object on the bottom panel: If this option is enabled then the details (name, size, attributes etc.) of any file/folder you select in the File Browser will be displayed in the File Browser information area. If this option is disabled, the File Browser information are will display the details of the directory you are browsing and would simply not respond to file/folder selections.

e)     Show extended properties panel: This option controls whether the extended properties panel is displayed or not. If this option is disabled, then the extended properties panel will not be displayed and the preview functionality will also be unavailable as it is a part of the extended properties panel.

i)       Automatically play video files: This option is only available if the “Show extended properties panel” is enabled. If enabled, it causes the inbuilt previewer to automatically start playing when you select a video file. If it is disabled, the file will be loaded but will not be played until you click the play button.

ii)    Automatically play audio files: This option is only available if the “Show extended properties panel” is enabled. If enabled, it causes the inbuilt previewer to automatically start playing when you select an audio file. If it is disabled, the file will be loaded but will not be played until you click the play button.

f)      Use the following color for encrypted and compressed files: Using this option, you can specify a color that the File Browser uses to represent encrypted and compressed files. This option is provided to maintain similarity with Windows Explorer.

 

3.     The Details View tab: This tab allows you to configure which columns are displayed when the File Browser is in Details view. The columns are divided in three categories namely: File Properties, Audio/Video Properties, and Document Properties.
