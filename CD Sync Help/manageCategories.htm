Managing Image Categories

 

With the category manager, you can add, edit, or remove existing image categories. To open the category manager, select Tools > Image Categories or press Ctrl+T. The category manager dialog box will be displayed:

 

 

Using the category manager, you can perform the following functions:

 

1.     Add new category: To add a new category, click the Add New button. A dialog box as shown in the following screen shot will be displayed:

Enter the details and select Ok in the dialog box to create the new category.

 

2.     Modify an existing image Category: To change the name and/or description of an existing category, select the desired category in category manager, and click the Modify button. A dialog box as shown in the following screen shot will be displayed:

Here you can change the name and description of the category as desired. When you are done, click Ok.

 

3.     Delete an existing image category: To delete an existing image category, select it from the list of the available categories, and click the Delete button.
