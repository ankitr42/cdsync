The Menu Bar and the Toolbar

 

The CD Sync Portable menu bar contains the following menus.

 

1.     The File menu: The file menu contains commands for working with the objects shown in the File Browser. Most of the commands in this menu are activated only when at least one item are selected in the File Browser. This menu contains the following commands:

a)     Open: If the item selected in the File Browser is a folder, then it will be opened in the File Browser. If the selected item is a file, then it will be opened (if the source media is available) with the program associated with the type of the file.

b)     Explore: This option is only available if the selected item is a folder. Clicking on this menu item will open (if the source media is available) the folder for browsing in Windows Explorer.

c)     Rename: Allows you to rename the selected item in-place.

a)     Edit Description: Opens a dialog box to allow you to add or edit the description of the selected item. For more details, see Editing File/Folder Descriptions.

b)     Delete: Used to delete the selected items. These items will be removed from the image.

c)     Item Properties: Opens the properties dialog box for the selected item(s). For more details, see File, Folder and Image Properties Dialog Boxes.

d)     Exit: I don’t think you need a description of this commandJ.

 

2.     The Image menu: This menu contains commands for creating new images, and working with existing images. This menu commands the following commands:

a)     New Image: This option allows you to create new images. Selecting this option will open up the New Image dialog box which you can use to create new images. For more details, see Creating New Images.

b)     Delete Image: This option is only activated when an image is selected. Use it to delete the currently active image.

c)     Image Properties: This option is also only activated when an image is selected. This option opens up the Image Properties dialog box, where you can view statistical information about that image and also change the path of the source media of the image. For more details, see File, Folder and Image Properties Dialog Boxes.

 

3.     The View menu: This menu contains a single sub-menu that contains commands to alter the appearance of the File Browser. It contains the following commands:

a)     Back: Go back to the last folder you were browsing.

b)     Forward: Go forward.

c)     Up: Go up one level to the parent folder.

d)     Root: Navigate back to the root folder of the image.

e)     Refresh: Refreshes the File Browser, the Extended Properties panel, the Image Information area and the File Browser information area.

f)      Icons: Displays objects with large icons.

g)     List: Displays objects as a list with small icons.

h)     Details: Displays objects in a tabular format with file details shown in columns.

 

4.     The Tools menu: This menu contains the following commands:

a)     Search: Opens up the Search Results window, where you can perform a Basic Search or an Advanced Search. The search result window also displays the search results. For more information, see Searching for Files/Folders.

b)     Export Images: Opens up the Export Image dialog box which you can use to export your images. See Importing and Exporting Images for more information.

c)     Import Images: Opens up the Import Images dialog box which you can use to import images. See Importing and Exporting Images for more information.

d)     Export Filelist: Opens up the Export Filelist dialog box which you can use to export the file list for the current image. See Exporting File List for more information.

e)     Image Categories: Opens up the Image Categories dialog box that allows you to create, delete, and edit existing image categories. See Managing Image Categories for more information.

f)      Options: Opens up the Options dialog box where you can set various application options. See Configuring Application Options for more information.

 

5.     The Help menu: This menu contains the following commands:

a)     Help: Opens up this help file.

b)     About: Displays information about the program version, developer, and credits.

 

The CD Sync Portable toolbar contains shortcut buttons for various frequently used menu items for quick access. The toolbar contains the following commands:

 

 

1.     New: Opens the New Image dialog box.

2.     Delete: Deletes the currently active image.

3.     Back: Browses back to the last folder in the browsing chain.

4.     Forward: Browses forward to the next folder in the browsing chain.

5.     Up: Browses one level up to the parent folder.

6.     Root: Browses back to the root folder of the image.

7.     Refresh: Refreshes the File Browser, the Extended Properties panel, the Image Information area and the File Browser information area.

8.     Properties: Opens the properties dialog box for the selected item(s).

9.     Search: Opens the Search Results window.

10.View: Provides options to change the way objects are displayed in the File Browser.

11.Options: Opens the Options dialog box.
