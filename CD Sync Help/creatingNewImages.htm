Creating New Images

 

The New Image dialog box is used to create new images. To access the New Image dialog box you can either select Image > New Image, or press Ctrl+N, or click the New button on the toolbar. The New Image dialog box has three tabs namely General, File Filtering, and Folder Filtering.

 

1.     The General tab: The General tab contains settings to specify the general options for the creation of the new image. It has the following options:

 

 

a)     Source path: Specify the path of the source media here.

b)     Picture path: Specify a picture to represent this image. This picture will be shown below the image browser whenever you select the image.

c)     Select category: Select the category in which you’d like to put your image. You can also create a new category by clicking the Create New button.

d)     Image Name: If you have used the browse button to select the image source, this place will probably already be filled. If the selected source is a folder, then the name of the folder is automatically filled in this box. If the selected source is the root directory of some drive, then the volume label of the drive is automatically filled in this box. However, you can change it to anything you want.

e)     Description: Here, you can type in a description for your image. Image description will be shown as a tool tip when you hover your mouse over the node representing the image in the image browser.

f)      Extended file information: This area allows you to select what extended information is copied about each file. There are four checkboxes, that allow you to select the type of information that will be copied:

i.       Copy file’s version information: Enabling this checkbox will copy version information for files that contain this information. Files such as executables, DLLs etc contain version information. The following elements are copied under this category: comments, company name, copyright, description, product name, major version, minor version, revision, major program version, minor program version, and program revision.

ii.    Copy file’s audio and video information: Enabling this checkbox will copy audio, music, and video information from audio/video files. It also copies the dimensions of image files. The following information is will be copied if an A/V/P file contains it: artist (if multiple artists are listed, then only the first one is copied), album, album year, title, track number, duration, genre, bit rate, sample rate, width, and height.

iii.  Copy document information: Enabling this will copy the following details from files that contain them: document author, document category, comments, company, document title, no. of pages, and subject.

iv.   Copy file icons: If selected, it will copy icons for .exe, .scr, .cur, and .ico files.

 

2.     The file filtering tab: This tab contains options that allow you to set filtration criteria for files. A file must satisfy all the constraints specified on this tab to be included in the image. Files that are found to be violating any of the filtration criteria are not included in the image. This tab contains the following options:

 

 

a)     Date filter: Use the date filter to filter out files with dates not belonging to the configured range. Note that you can set ranges for all three types of dates, namely created, modified, and accessed date.

b)     File type filtering: Use this option to filter out files based on their extension. There are two options: You can either choose to include files with only specific extensions, or you can exclude files that have a specific extension. Select the appropriate radio button and enter the file extensions in the textbox provided. You can use semicolon to separate multiple file extensions.

c)     Size filtering: Use this area to configure file size filtering. You can set a limit on the size of the file to be included in the image. Enter the lower and upper bounds in the appropriate boxes. Remember that the unit used here to measure file size is KBs.

d)     Attributes filtering: You can use this area to configure filtering of files based on their attributes. Selecting any attribute here will exclude any file with that attribute from the image.

e)     Exclusion list: The ‘Exclude these files’ list is used to specifically specify the files that you do not want to be included in the image. Use the Add, Remove and Remove All buttons to add/remove files from this list.

 

3.     The folder filtering tab: The folder filtering tab contains a list box which you can use to create a list of folders that you don’t want to be included in the image. It is depicted in the following screen shot:

 

 

a)     Exclusion list: Add those folders to the exclusion list that you don’t want to be included in your image. Note that adding a folder to this list means that neither the folder itself, nor its contents will be added to the image. Use the Add, Remove, and Remove All buttons to add/remove entries.

b)     Search depth: Use the up-down box to configure, how deep the program will scan the image source. For example, it you select 0 here, only the source folder will be scanned, and none of its subdirectories will be scanned. Specify -1 to scan all subfolders.
