Editing File/Folder Descriptions

 

CD Sync Portable allows you to copy file descriptions while creating a new image. Later, you can also edit/remove file description using the Edit Description dialog box. You can assign a custom description to each and every file/folder in your image. The Edit Description dialog box enables you to edit existing description text for a file / folder, or add new one. You can add a description of length 1024 characters at a maximum. The label below the textbox shows the number of characters still available.

To access the Edit Description dialog box, first select the file/folder for which you want to edit/add the description and then select File > Edit Description. Alternatively, you can right click the desired file/folder and select Edit Description from the shortcut menu that appears.

 

 

The dialog box contains a large text in which you type the description you want to set. As you type, you are notified of the number of characters that are remaining. When you are done, click OK and the description will be saved to the database.
