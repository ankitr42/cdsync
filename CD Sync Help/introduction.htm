Introduction to CD Sync Portable

 

Before we understand what this program is, and what it does, it’d be good for both you and me if we just go through these questions once.

·        Do you have a huge CD/DVD collection, containing your music, movies, documents and other stuff?

·        Are you fed up of poppin’ in about dozen of discs into your optical drive, just to see if the disc contains the file you’re looking for?

·        Do you spend less time dancing and more time straining your brain, to recall where did you put that Bryan Adams album, over the weekend party organized by you?

·        Or, Are you the one who’s just not happy with the speed and abilities of Windows file search facility?

 

If any of these problems sound familiar to you, then you’ve got the right tool to kiss ‘em all goodbye.

 

Formal description

CD Sync Portable is a disc cataloging software that allows you to catalog your CDs/DVDs/Flash Drives/Hard drive Folders etc. A catalog in the terminology of CD Sync Portable is called an ‘Image’ and from now onwards, whenever we refer to ‘Image’, it should be taken to mean a disc/folder catalog. With CD Sync Portable, you get the ability to browse and search your data, even when the source media is not available.

CD Sync Portable provides several predefined categories under which you can categorize images you create. You can also create new categories as and when you desire. CD Sync Portable offers powerful search facilities, which allow you to search for a file/folder using basic file properties (like filesize, attributes, etc), as well as advanced attributes like Artist name, Title, Description, Company name etc. All these capabilities enable you to perform content specific searches.

Some of the major features of CD Sync Portable are:

 

·        Portability. This means that you can carry your whole catalog wherever you go. Just unzip CD Sync Portable in a flash drive or external HDD and start creating portable catalogs that you can carry with you.

·        Stores disks' and folders' file structure as it is, to allow you to browse and search them even when they’re not available.

·        Several image categories (like Games, music movies etc.) are provided, so that you can categorize your images. You can also create new categories.

·        Stores extended properties about each file. CD Sync Portable can store above 25 types of properties about each file. This includes:

ü      Standard file properties, like name, size, and date(s) etc.

ü      Music properties, like artist name, album title, album year, and song title etc.

ü      Document properties, like company, subject, and category etc.

ü      Video and image properties, like width and height.

ü      File icons for .ico, .exe, .scr, .cur files.

·        Allows you to set custom description for each and every file/folder in your image.

·        Incorporates powerful searching features that allow you to search any number of images at a time. Also features content specific search capability (for ex. You can search your music collection for all songs by ‘Linkin Park’.)

·        Allows you to export your images, in case you’re migrating to new PC and don’t want to create all those images again. This feature also comes in handy in case of data loss; you can always restore your images from the backup copy.

·        Allows you to export the file-list for an Image in XML format. From there, it can be read and interpreted by any other program.

·        Features a clean and friendly Windows Explorer like interface. So that not too much of your time is wasted recalling where a particular command was located.

·        If the source media of an Image is available, the built-in previewer allows you to preview almost all major types of audio, video, and image files contained in the image. The built-in previewer can also be configured to automatically play audio and video files as you select them in the file browser.
