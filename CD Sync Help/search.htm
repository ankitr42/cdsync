Searching for Files/Folders

 

CD Sync Portable comes with very powerful search capabilities. CD Sync Portable allows you to search for your files/folders in multiple images at a time using many advanced file/folder properties like title, album, company, file size, description etc. The search capabilities of the program can be exploited using any of the two interfaces: Basic search and Advanced search. The basic search provides a simple interface where you specify a filename (wildcards allowed) and simply select the images on which you want to execute your search query, and you’re ready to go. The advanced search has been designed to perform much more advanced and sophisticated search queries. For instance, you can search for all mp3 files sung by Britney Spears in the year 2001.

To begin a search, while in the main window, click the Search button on toolbar, or press F3, or select Tools > Search. The Search Results window will appear:

 

 

The Search Results window is the both the starting point & end point for your search. The window contains a toolbar that contains buttons to control its functioning. The toolbar contains the following buttons:

1.     Basic Search: Launches the Basic Search dialog box.

2.     Advanced Search: Launches the Advanced Search dialog box.

3.     Open: Only active when an item is selected. Clicking on this button will open the selected item with its associated program.

4.     Explore in main window: Only active when an item is selected. Clicking this button will activate the main program window, and browse to the directory containing item in the File Browser.

5.     Properties: Only active when an item is selected. Opens the property dialog box for the selected item.

6.     Extended properties: Toggles the extended properties panel on and off.

7.     Autoplay: Only active when the extended properties panel is visible. Toggles automatic playing of audio/video files in the previewer on and off.

8.     View: Changes the current display style of the search results.

 

Performing a basic search

When you click the Basic Search button on the toolbar, you will be presented with the Basic Search dialog box.

The basic search dialog box is simple and has very little to be explained. A list of all the images is displayed in the form of a tree structure from which you can select the images to be searched. The textbox just above the image tree is the place to put your search query. You can specify multiple filenames in the box, separating each pair with a ‘|’ sign. Wildcards are also allowed. The dialog box also contains a link to the Advanced Search dialog box. When you are ready, hit the Search button to begin the search.

 

 

 

 

 

 

 

 

Performing an advanced search

If you are not happy with the control provided by basic search, you can perform an advanced search. To launch an advanced search, click the Advanced Search button on the toolbar. The Advanced Search dialog box will be shown:

 

 

The advanced search dialog provides lots of options to customize your search query. You can utilize over 20 file properties to perform the search. The right half of the dialog box provides the list of images from which you can select the images that you would like to search. The left half contains various options that you can customize. A detailed description of each option follows:

1.  File name: This text box allows you to enter the name of the file/folder you want to search. You can specify multiple filenames, separating each one with the pipe ( | ) sign.

2.  Search for: This option specifies whether you want to include only for files, or only for folders, or for both in the search results.

3.  General properties: This group contains options to configure general file properties like filesize etc.

a)     File size: Enable this checkbox to filter search results based on filesize. Using this option, you can set an upper bound, or a lower bound, or a range, or an exact figure that a file must match to be included in the search results.

b)     Created date: Enable this checkbox to filter search results based on the creation date. Like filesize, you can set an upper bound, a lower bound, a range or an exact figure.

c)     Accessed date: Enable this checkbox to filter search results based on the access date of the file. Like filesize, you can set an upper bound, a lower bound, a range or an exact figure.

d)     Modified date: Enable this checkbox to filter search results based on the modification date of the file. Like filesize, you can set an upper bound, a lower bound, a range or an exact figure.

e)     File attributes: This checkbox allows you to introduce filters based on file attributes. Selecting file attributes here requires a file/folder to have all those attributes to be included in the search results.

4.  Music and audio properties: This section contains options to perform filtering based on musical or audio data such as bitrate, sample rate, song title etc.

a)     Title: The title of the song (or document) to search for. The two radio buttons to the right of the textbox let you select how data in the textbox is matched against the file’s information. For ex., if you have selected ‘Exact’, only those files will be included whose title matches exactly with the one specified in this textbox. On the other hand, selecting ‘Substring’ will also include those files whose title contains the one specified in this textbox as a substring.

b)     Artist: Artist information of the song to search for. Substring or exact matching allowed.

c)     Album: The album to which the song belongs. Substring or exact matching allowed.

d)     Genre: The genre of the song. Substring or exact matching allowed.

e)     Bitrate: The bitrate of the audio file. You can specify the bitrate either as a lower bound, an upper bound, a range, or an exact value. Bitrate is specified in Kbps.

f)      Sample rate: The sample rate (sample / second) of the audio file. Sample rate can also be specified either as a lower bound, an upper bound, a range, or an exact value. Sample rate is specified in Hz.

5.  Document properties: This section contains options to perform search based on document information such as document author, company name etc.

a)     Description: The document description to match. Substring or exact matching allowed.

b)     Author: The author of the document to look for. Substring or exact matching allowed.

c)     Subject: The subject of the document. Substring or exact matching allowed.

d)     Category: The category of the document. Substring or exact matching allowed.

e)     Comments: The comments on the document. Substring or exact matching allowed.

f)      Company: The Company of the document (or DLL/exe). Substring or exact matching allowed.

 

Search results

When you click the search button in any of the search dialog boxes, and the search will begin. While the search is active, you will see a Stop Search button on the search results window toolbar.

 

 

When the search completes, the results will be displayed in the search results window. Search results are displayed in groups based on the images that they belong to:

 

 

Interacting with the search results window is similar to interacting with the main program window. When you select an item in the results window, its extended properties (if available) will be displayed in the extended properties panel, also if the source media of the image to which the item belongs is available, the integrated previewer will generate a preview (if the item is previewable).
